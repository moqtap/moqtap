## [0.1.0] - 2026-03-21

Initial release targeting MoQT draft-ietf-moq-transport-14.

### Added

- `moqtap subscribe` — connect and subscribe to a track with optional trace capture
- `moqtap fetch` — connect and fetch a track range
- `moqtap trace` — read and display .moqtrace files (text/JSON) with optional metrics
